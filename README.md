# swift-units
Unit conversion Swift library, with other relevant information.

## Requirements
- Swift >= 5.1
- MacOS >= 10.15
- iOS >= 13.0

## Current features
### Units
- `AccelerationUnit`
- `ActionUnit`
- `DensityUnit`
- `ElectricChargeUnit`
- `EnergyUnit`
- `ForceUnit`
- `FrequencyUnit`
- `LengthUnit`
- `MassUnit`
- `PressureUnit`
- `SpeedUnit`
- `TemperatureUnit`
- `TimeUnit`
